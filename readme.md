# 🧠 Классификация рукописных цифр MNIST с PyTorch Lightning
Проект демонстрирует обучение полносвязной нейронной сети для распознавания рукописных цифр из датасета MNIST. Код написан с использованием фреймворка PyTorch Lightning, что обеспечивает чистую структуру и автоматизацию рутинных задач обучения.

# 📌 Основные возможности
* PyTorch Lightning: Автоматизирует циклы обучения, валидации и тестирования.
* Автоматический выбор устройства: Поддерживает CPU и GPU «из коробки».
* Сохранение моделей: Фиксирует лучшую эпоху на основе точности валидации (val_acc).
* Анализ ошибок: Извлекает неверно классифицированные изображения и сохраняет их в отдельные папки с текстовыми метками.
* Матрица ошибок: Визуализирует качество работы модели по всем классам.

# 🛠️ Установка
Перед запуском убедитесь, что у вас установлены все необходимые библиотеки. Выполните следующие команды:
```
pip install torch torchvision
pip install pytorch-lightning torchmetrics
pip install matplotlib seaborn pillow
```

# 📂 Структура проекта
* LitMNIST — класс модели (Молния-модуль), содержащий архитектуру и шаги обучения.
* MNISTDataModule — модуль инкапсуляции данных (скачивание, аугментация, загрузчики).
* ./data — папка со скачанным датасетом MNIST.
* ./my_saved_models — директория с лучшими чекпоинтами моделей.
* ./error_images — папка с ошибочно распознанными изображениями (сортировка по подпапкам вида true_X_pred_Y).

# 🚀 Как использовать
1. Обучение и тест
Просто запустите весь скрипт. По умолчанию зафиксирован сид воспроизводимости (seed_everything(42)), а обучение длится 2 эпохи.
2. Результаты работы
После выполнения скрипта:
	* Модель достигнет определенной точности на тестовой выборке.
	* В папке ./error_images появятся изображения, на которых модель ошиблась. На них будет написано T (True — истина) и P (Predicted — предсказано).
	* Появится интерактивное окно с визуализацией матрицы ошибок (Confusion Matrix).

# 📊 Пример визуализации матрицы ошибок
После завершения тестирования генерируется хитмап для наглядного поиска слабых мест модели:
* Ось X: Предсказанный класс.
* Ось Y: Истинный класс.
